- **Zero Balance Displayed**:
  - Verify if any recent transactions were posted.
  - Check if pending transactions have temporarily reduced balance.
  - Confirm account is not in dormancy.
  - Ensure system is not undergoing maintenance.

- **Unable to View Balance**:
  - Clear app cache (mobile app users).
  - Update to the latest version of the banking app.
  - Try switching between NetBanking and Mobile Banking.

- **Mismatch Between ATM Slip and NetBanking Balance**:
  - Allow 24 hours for system sync.
  - Contact branch if discrepancy persists.
