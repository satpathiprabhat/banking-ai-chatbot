# Fund Transfer Issues

- **Transfer Failed**:
  - Confirm beneficiary details (account number, IFSC) are correct.
  - Check if daily transaction limit has been exceeded.
  - Retry after ensuring no ongoing maintenance.

- **NEFT/IMPS Delays**:
  - NEFT transactions may take up to 2 hours.
  - IMPS should reflect instantly; if delayed, check for downtime alerts.

- **UPI Transaction Declined**:
  - Ensure UPI PIN is correct.
  - Verify linked account has sufficient funds.
  - Confirm UPI app is updated to the latest version.
